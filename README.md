# Bright_TV_CASE_Study

Case Study Overview
BrightTV is a subscription-based streaming platform looking to grow its user base and increase engagement over the current financial year. This case study analyzes user profile data alongside session-level viewership activity to generate insights that support the Customer Value Management (CVM) team. By converting all timestamps to South African local time, the analysis identifies key usage patterns across different times of day and days of the week, enabling a clearer understanding of when and how users consume content.

The study focuses on user demographics, viewing behavior, and content preferences to uncover the main drivers of consumption. It also highlights opportunities to improve engagement during low-activity periods through targeted content strategies. Based on these insights, recommendations are provided to help grow the subscriber base, including personalized experiences, targeted marketing efforts, and initiatives designed to improve both user acquisition and retention.
