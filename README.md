# Bright_TV_CASE_Study

Case Study Overview
BrightTV is a subscription-based streaming platform looking to grow its user base and increase engagement over the current financial year. This case study analyzes user profile data alongside session-level viewership activity to generate insights that support the Customer Value Management (CVM) team. By converting all timestamps to South African local time, the analysis identifies key usage patterns across different times of day and days of the week, enabling a clearer understanding of when and how users consume content.

The study focuses on user demographics, viewing behavior, and content preferences to uncover the main drivers of consumption. It also highlights opportunities to improve engagement during low-activity periods through targeted content strategies. Based on these insights, recommendations are provided to help grow the subscriber base, including personalized experiences, targeted marketing efforts, and initiatives designed to improve both user acquisition and retention.

Tools Used to do the CAse Study
To effectively deliver this case study, a combination of analytical, planning, and visualization tools was used to ensure a structured and insight-driven approach. Miro was used at the initial stage to design a clear process flow and map out the analytical framework, helping to visualize the end-to-end approach from data ingestion to insight generation. Project planning and timeline management were handled using Canva, where a Gantt chart was developed to track key milestones, manage deliverables, and ensure the project stayed on schedule.

For data processing and transformation, Databricks was utilized, leveraging SQL to clean, join, and aggregate large datasets efficiently. Microsoft Power BI was used to build interactive dashboards and visualizations that highlight key trends and insights. Finally, Claude.ai was also used to design and structure the final presentation, ensuring that insights were communicated clearly, visually, and in a professional format suitable for stakeholders.

Key Insights 
82% of registered users were active viewers in Q1 2016, generating 10000 sessions across 21 channels. The remaining 989 dormant users represent an immediate win-back opportunity for the CVM team. March drove 48% of all sessions, largely powered by ICC Cricket World Cup 2011 replays on SuperSport.
